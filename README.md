## ğŸŒŸ Smart Stars Assessment

The **Smart Stars Assessment** is a dynamic challenge that picks 10 random questions from a pool of 30+. It evaluates the child's progress in logic, pattern recognition, counting, and emotional understanding.
- **Dynamic Questioning**: Every session is different to maintain engagement.
- **Detailed Feedback**: Provides encouraging explanations for every answer.
- **Star Rewards**: Kids collect stars as they progress, building confidence!

---

## ğŸŒˆ How It Helps Kids with Autism

This project was built with a "sensory-first" approach:

1.  **Sensory-Friendly Design**: Uses a calming **Pastel Color Palette** and avoids high-contrast or flashing elements that can cause sensory overload.
2.  **Social-Emotional Learning (SEL)**: The *Emotion Garden* specifically helps children identify and process different emotions through interactive storytelling.
3.  **Clear & Simple HUD**: Minimalistic UI ensures that the child focuses only on the core activity without being distracted by unnecessary buttons or ads.
4.  **Positive Reinforcement**: Every success is celebrated with visual rewards and encouraging text, fostering a growth mindset.
5.  **Predictable Patterns**: Logical game mechanics provide a sense of structure and routine, which is often comforting for kids with ASD.
